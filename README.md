# Helix Config

## This is my helix config that I use currently.

### Custom Theme:
  * transparent + gruvbox_dark_hard
  
### Snippets:
  * html_template!
  * cmake_template!

### How do I use snippets?
  1. press : key
  2. type insert-output
  3. press space key
  4. type the name of snippet
